# 📃 요구사항 명세 (Multi version)

- Home page
  - 구글로그인 버튼 및 랭킹버튼을 보여준다.
  - 로그인된 상태면 유저이름과 유저가 플레이한 게임횟수, 최고기록, 시작버튼, 로그아웃 버튼을 보여준다.
 - Ranking page 
  - 랭킹을 제공한다. 1~30등정도까지 볼수 있다.
  - 자신의 랭킹은 볼 수 있게 한다.
